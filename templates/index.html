Here is the Manual and initial starting instructions for the Interface for Research Opportunity program, designed for REU program adminstration.

Set Up

First steps first, to start setting up the program, you need to create admin users. To do that, follow these steps:

  1. Open up a terminal, command prompt, or similar program.
  2. Type 'python'
  3. Type 'manage.py createsuperuser'
  4. Follow the prompts onscreen
  5. Log into the admin interface at base-url/admin/login (e.g. spur.uoregon.edu/admin/login)
